
# Lotto Number Generator

## Overview

This project is a simple web application that generates random lottery numbers. It's built using modern web standards, including HTML, CSS, and JavaScript, and leverages Web Components for a modular and maintainable structure.

## Design and Features

### Layout and Style

*   **Responsive Design:** The application is fully responsive and works on both mobile and desktop devices.
*   **Color Palette:** A vibrant and energetic color palette is used to create a visually appealing experience.
*   **Typography:** Expressive typography is used to create a clear visual hierarchy.
*   **Interactivity:** Interactive elements have a "glow" effect to provide visual feedback to the user.

### Features

*   **Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Web Components:** The lottery number display is encapsulated in a Web Component for reusability.
